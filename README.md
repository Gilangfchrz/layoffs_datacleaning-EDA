# Proyek Analisis PHK Global

Repositori ini berisi proyek lengkap analisis data PHK (layoffs) secara end-to-end, menggunakan data nyata. Proyek dibagi menjadi dua tahap utama:

## 1️⃣ Tahap 1: Pembersihan Data (Data Cleaning)
- Menggunakan MySQL untuk:
  - Menghapus duplikat
  - Menangani nilai NULL
  - Menstandardisasi format tanggal dan kategori


## 2️⃣ Tahap 2: Analisis Eksploratori Data (EDA)
- Menggunakan Python (Pandas & Matplotlib):
  - Analisis tren PHK berdasar industri, negara, dan timeline
  - Visualisasi insight penting


## 🧰 Teknologi & Tools yang Digunakan
- MySQL & MySQL Workbench  
- Python (Pandas, Matplotlib, Seaborn)  
- Jupyter Notebook
